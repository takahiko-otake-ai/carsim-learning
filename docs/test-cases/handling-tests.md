# 操縦安定性テスト

`Handling` カテゴリには、車両の操縦安定性（ハンドリング）を評価するための包括的なテストケースが収録されています。標準的な試験法に準拠したシナリオから、カスタム評価まで幅広い用途に対応しています。

## 標準テストの種類

### Double Lane Change（ISO 3888-1）

ダブルレーンチェンジテストは、車両の**過渡的な操縦応答**を評価する代表的な試験です。パイロンで構成されたコース内を、指定速度で車線変更しながら走行します。

### 定常円旋回（Steady-State Circular Driving）

一定半径の円を等速で旋回し、**アンダーステア勾配**を測定します。速度を段階的に上げることで、線形領域から非線形領域への遷移を評価します。

### ステップステア（Step Steer）

直進走行中にステアリングを急速に操舵し、車両の**過渡応答特性**を評価します。ヨーレートと横加速度の立ち上がり時間、ピーク値、定常値を確認します。

### サインウィズドウェル（Sine with Dwell）

正弦波操舵の途中で操舵角を保持する入力パターンです。**ESC（横滑り防止装置）**の評価に使用され、FMVSS 126 で規定されています。

### J ターン

直進から一方向への急操舵を行い、車両の**ロールオーバー傾向**を評価します。SUV やピックアップトラックなど重心の高い車両で重要なテストです。

### フィッシュフック（Fishhook）

急操舵と逆操舵を連続して行い、**ロールオーバー耐性**を評価します。J ターンより厳しい条件であり、車両の限界挙動を確認します。

### ランプステア（Ramp Steer）

操舵角を時間に対して一定勾配で増加させ、**定常応答特性**を連続的に取得します。アンダーステア特性の評価に適しています。

### 周波数応答（Frequency Response）

異なる周波数の正弦波操舵を入力し、ヨーレートや横加速度の**振幅比**と**位相遅れ**を評価します。車両の動的応答帯域を把握できます。

## 主要パラメータ

| パラメータ | 説明 | 設定例 |
|-----------|------|--------|
| 車速 | テスト中の走行速度 | 60–120 km/h |
| 操舵入力タイプ | オープンループ / クローズドループ | テストにより選択 |
| 操舵角振幅 | ステアリングホイール角度 | 30–270 deg |
| 操舵速度 | 操舵角の時間変化率 | 100–500 deg/s |
| 路面摩擦係数（μ） | 路面のグリップレベル | 0.3–1.0 |

:::note オープンループとクローズドループ
**オープンループ**テストでは操舵入力が時間関数で定義され、車両応答に依存しません。再現性が高く、車両間比較に適しています。**クローズドループ**テストでは仮想ドライバーが経路を追従するよう操舵を行い、実走行に近い条件を再現します。
:::

## 評価指標

操縦安定性テストで確認すべき主要な評価指標は以下の通りです。

- **横加速度（Lateral Acceleration）** — コーナリング中の横方向力
- **ヨーレート（Yaw Rate）** — 車両の回転角速度
- **ロール角（Roll Angle）** — 車体の横傾斜角度
- **横すべり角（Sideslip Angle）** — 車両重心の進行方向と車両向きの差
- **アンダーステア勾配（Understeer Gradient）** — 定常旋回特性の指標
- **応答時間（Response Time）** — 操舵入力から出力ピークまでの時間

## 車両比較の方法

CarSimでは、同じ Procedure と Road を使用して異なる Vehicle Assembly のシミュレーション結果を重ね合わせて比較できます。

1. **ベース車両でシミュレーション実行** — 結果を保存
2. **Vehicle Assembly を変更** — 比較対象の車両に切り替え
3. **同一条件で再実行** — Procedure と Road は変更しない
4. **Overlay でグラフ比較** — 両方の結果を同一グラフ上にプロット
5. **Set Color で色分け** — 各車両の結果を異なる色で表示し、識別しやすくする

:::tip 効率的な比較
複数車両を体系的に比較する場合は、**Batch Run**（一括実行）機能が便利です。比較したいすべてのテストケースを登録し、一度に実行できます。
:::

## 規制基準との関係

操縦安定性テストの多くは、国際的な安全基準に基づいています。

| 規格 | 内容 | 対応テスト |
|------|------|-----------|
| **FMVSS 126** | ESC（横滑り防止装置）性能要件 | サインウィズドウェル |
| **ISO 3888-1** | ダブルレーンチェンジ試験方法 | DLC テスト |
| **Euro NCAP** | ESC 性能評価プロトコル | サインウィズドウェル、J ターン |
| **ISO 4138** | 定常円旋回試験方法 | 定常円旋回 |

CarSimのデータベースにはこれらの規格に準拠した Procedure が含まれているため、規制要件への適合評価をシミュレーション上で効率的に実施できます。

:::info
規制テストのパラメータ（速度、操舵角、判定基準など）は規格文書で厳密に定義されています。CarSimのサンプルは一般的な条件を再現していますが、正式な適合評価では最新の規格要件を確認してください。
:::
