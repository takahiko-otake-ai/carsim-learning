# Quick Start ガイド例

`Quick_Start` カテゴリには、CarSimを初めて使うユーザー向けの基本的なシミュレーション例が収録されています。これらの例を通じて、Run Control の操作方法、データセットの構成、シミュレーション結果の確認方法を学ぶことができます。

## 収録されているシナリオ

### Baseline：基本走行シナリオ

最も基本的なテストケースです。デフォルトの **C-Class ハッチバック**車両が、標準的な道路上を**クローズドループドライバー**で走行します。

- **Vehicle Assembly**：C-Class, Hatchback（標準的な乗用車パラメータ）
- **Procedure**：定速走行、指定経路追従
- **Road**：平坦な直線道路

このシナリオは、CarSimが正しくインストールされ動作しているかの確認にも使用できます。

### Double Lane Change（DLC）テスト

ISO 3888-1 に準拠した**ダブルレーンチェンジ**テストです。複数の速度条件が用意されています。

| テストケース | 速度 | 主な評価項目 |
|------------|------|-------------|
| DLC 60 km/h | 60 km/h | 基本的な操縦性確認 |
| DLC 80 km/h | 80 km/h | 標準的な性能評価 |
| DLC 120 km/h | 120 km/h | 限界域での安定性確認 |

DLC テストでは、車両の**横方向応答**、**ヨーレート**、**ロール角**などをプロットで確認できます。

### 加速テスト

停止状態からのフル加速を評価するテストケースです。

- **Procedure**：スロットル全開（WOT: Wide Open Throttle）
- **評価項目**：0-100 km/h 加速時間、エンジン回転数、車速プロファイル
- **Road**：平坦な直線道路、高摩擦路面

### 定速走行テスト

指定された速度での直進走行を行うシンプルなテストケースです。定常状態での車両挙動を確認する際のベースラインとして使用します。

## 各コンポーネントの設定内容

Quick Start の各テストケースでは、以下の主要コンポーネントが設定されています。

**Vehicle（車両）**
- 車両質量、ホイールベース、トレッド幅
- サスペンション特性（ばね定数、減衰力）
- タイヤモデル（Magic Formula パラメータ）

**Procedure（手順）**
- ドライバーモデル（オープンループ or クローズドループ）
- 目標速度プロファイル
- 操舵入力パターン

**Road（道路）**
- 道路形状（直線、カーブ、車線変更コース）
- 路面摩擦係数（μ）
- 路面勾配

## Quick Start 例のカスタマイズ

既存の Quick Start 例を基に、独自のテストケースを作成する手順を紹介します。

1. **データセットのコピー** — 変更したいテストケースを選択し、Copy ボタンで複製
2. **車両パラメータの変更** — Vehicle Assembly を開き、車両質量やサスペンション設定を調整
3. **走行条件の変更** — Procedure の速度や操舵入力を目的に合わせて変更
4. **道路条件の変更** — Road の摩擦係数や形状を変更して異なる環境を再現
5. **シミュレーション実行と結果確認** — Run Math Model で実行し、結果をプロットで比較

:::tip 初心者向けアドバイス
パラメータの変更は**一度に1つずつ**行うことを推奨します。複数のパラメータを同時に変更すると、どのパラメータがどの結果に影響を与えたか判別しにくくなります。
:::

:::note
Quick Start 例はシンプルな構成になっているため、CarSimの各機能を理解するための出発点として最適です。より高度なテストケースについては、`Handling` や `Brake` カテゴリの例を参照してください。
:::
