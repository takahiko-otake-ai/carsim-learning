# ブレーキテスト

`Brake` カテゴリには、車両の制動性能を評価するための各種テストケースが収録されています。直進制動からABS評価、旋回制動まで、制動システムの設計・検証に必要なシナリオが網羅されています。

## ブレーキテストの種類

### 直進制動（Straight-Line Braking）

最も基本的なブレーキテストです。直線路上で定速走行から制動を行い、停止距離やピーク減速度を評価します。

### 旋回中制動（Braking in Turn）

コーナリング中にブレーキを入力した際の車両安定性を評価します。旋回半径や車速を変えて、制動時の**ヨーレート変化**や**スピン傾向**を確認します。

### ABS 評価テスト

ABS（アンチロック・ブレーキ・システム）の制御効果を評価します。ABS ON/OFF を切り替えて比較することで、ホイールロック防止効果と操舵性維持を確認できます。

### ブレーキフェードテスト

繰り返し制動によるブレーキの**温度上昇**と**制動力低下**を評価します。連続した制動サイクルの中で、制動距離がどのように変化するかを確認します。

### 緊急制動テスト

高速走行からの急ブレーキ操作を再現します。AEB（自動緊急ブレーキ）システムの評価にも使用されるシナリオです。

## 主要パラメータ

| パラメータ | 設定項目 | 典型的な値 |
|-----------|---------|-----------|
| ブレーキ入力タイプ | M/C 圧力 or ペダル踏力 | 0–15 MPa / 0–500 N |
| ブレーキ配分 | 前後配分比 | 前 60–70% / 後 30–40% |
| ABS | ON / OFF | テストにより切替 |
| 路面摩擦係数（μ） | 乾燥 / 湿潤 / 凍結 | 0.9 / 0.5 / 0.2 |
| 初速 | 走行速度 | 60–120 km/h |

:::info ブレーキ入力について
CarSimでは、ブレーキ入力を **マスターシリンダー圧力**（M/C Pressure）と**ペダル踏力**（Pedal Force）の2種類で指定できます。外部コントローラーとの連携では、M/C 圧力入力が一般的です。
:::

## 評価出力

ブレーキテストで確認すべき主要な出力信号は以下の通りです。

- **停止距離（Stopping Distance）** — 制動開始から停止までの走行距離
- **減速度プロファイル（Deceleration Profile）** — 時間に対する減速度の推移
- **ホイールロック検出（Wheel Lock-up Detection）** — 各輪のスリップ率、ロック発生の有無
- **制動時ヨーレート（Yaw Rate During Braking）** — 制動中の方向安定性指標
- **ピッチ角（Pitch Angle）** — ノーズダイブの程度

## 典型的なテスト手順

標準的なブレーキテストは、以下の手順で構成されます。

1. **定速走行** — 指定速度に到達するまで加速し、安定走行を維持
2. **ブレーキ入力** — 指定のタイミングでブレーキを入力（ステップ入力またはランプ入力）
3. **完全停止 or 目標減速度** — 停止するまで、または目標減速度に到達するまで制動を継続

## カスタムブレーキテストの作成

既存のブレーキテストをベースに、独自のテストケースを作成する方法を紹介します。

1. **既存テストケースのコピー** — `Brake` カテゴリから目的に近いテストケースを選びコピー
2. **ブレーキパラメータの変更** — 制動力の大きさ、入力プロファイル（ランプ、ステップ）を調整
3. **路面摩擦の変更** — Road の μ 値を変更して、低μ路面での挙動を確認
4. **ABS ON/OFF 比較** — 同じ条件で ABS の有無を切り替えて結果を比較

:::tip 比較のコツ
同一テストケースでABS ON/OFF を比較するには、2つのデータセットを作成し、**Overlay** 機能で結果を重ね合わせると効果的です。`Set Color` を使えば、グラフの色を区別しやすくなります。
:::

路面摩擦を変えたテストでは、μ = 0.9（乾燥アスファルト）、μ = 0.5（湿潤路面）、μ = 0.2（凍結路面）のように段階的に条件を変えることで、制動システムの限界性能を体系的に評価できます。
