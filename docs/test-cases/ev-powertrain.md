# EV パワートレインテスト

`EV` カテゴリには、電動車両（EV）およびハイブリッド車両（HEV）のパワートレイン性能を評価するテストケースが収録されています。モーター特性、バッテリー管理、回生制動、エネルギー効率など、電動化に特有の評価項目を網羅しています。

## EV テストシナリオ

### 加速性能テスト

電動モーターの瞬時トルク特性を活かした加速性能を評価します。

- **シナリオ**：停止状態からの全開加速（0-100 km/h）
- **評価項目**：加速時間、モーター出力プロファイル、バッテリー放電電流
- **特徴**：ICE 車両との比較で、EV のトルク応答の速さを確認

### エネルギー消費テスト（EPA スタイル走行サイクル）

標準的な走行パターンでのエネルギー消費量を評価します。

- **シナリオ**：EPA 走行サイクル（都市走行、高速走行）を再現
- **評価項目**：電費（kWh/km または Wh/mile）、航続距離推定
- **Procedure**：クローズドループドライバーが速度プロファイルを追従

### 回生制動効率テスト

減速時のエネルギー回収効率を評価します。

- **シナリオ**：制動パターンを繰り返し実行し、回生量を測定
- **評価項目**：回生エネルギー量、回生率、バッテリー SOC 回復量
- **パラメータ**：回生ブレーキ配分、回生上限トルク

### バッテリー SOC モニタリング

走行中のバッテリー残量（State of Charge）の推移を監視します。

- **シナリオ**：長距離走行を模擬し、SOC の消費パターンを確認
- **評価項目**：SOC 推移、バッテリー温度上昇、電圧降下

## HEV テストシナリオ

### モード遷移テスト（EV ↔ HEV）

ハイブリッド車両における EV モードとエンジン併用モードの切り替え挙動を確認します。

- **評価項目**：遷移のスムーズさ、駆動力の連続性、エンジン始動ショック
- **トリガー条件**：SOC 閾値、要求トルク、車速

### 燃費評価テスト

HEV のトータル燃料消費量を評価し、エネルギーマネジメント戦略の効果を検証します。

## 主要パラメータ

| パラメータ | 説明 | 設定例 |
|-----------|------|--------|
| モーター特性 | トルク-回転数マップ、効率マップ | モーター種別により異なる |
| バッテリー容量 | 総エネルギー容量 | 40–100 kWh |
| バッテリー電圧 | 公称電圧 / 最大・最小電圧 | 350–800 V |
| 内部抵抗 | 充放電時の電気抵抗 | SOC・温度依存 |
| バッテリー温度 | 初期温度、熱モデル | 25°C（標準） |
| PMC 設定 | Power Management Control | 回生配分、出力制限 |

## パワートレインアセンブリの種類

CarSimでは、4種類の電動パワートレインアセンブリが用意されています。用途に応じて適切なタイプを選択します。

| アセンブリタイプ | 構成 | 用途 |
|-----------------|------|------|
| **Power Split** | エンジン + MG1 + MG2 + 遊星歯車 | THS 型 HEV（プリウス型） |
| **Parallel Hybrid** | エンジン + モーター（並列配置） | P2 型 HEV |
| **Series Hybrid** | エンジン + 発電機 + 駆動モーター | レンジエクステンダー EV |
| **Electric** | モーター（前軸/後軸/両軸） | BEV（純電動車両） |

:::tip アセンブリ選択のポイント
純粋な EV の評価には **Electric** タイプを使用します。HEV のエネルギーマネジメント戦略を検討する場合は、車両のアーキテクチャに合ったタイプ（Power Split、Parallel、Series）を選択してください。
:::

## 評価出力

EV/HEV テストで確認する主要な出力信号です。

- **モータートルク / パワー / 回転数** — モーターの動作点と出力状態
- **バッテリー SOC** — 残充電量の時間推移
- **エネルギー消費量** — 電費（kWh/km）または燃費
- **航続距離推定** — SOC 消費率から算出される走行可能距離
- **バッテリー電圧・電流** — 充放電の電気的挙動
- **回生エネルギー** — 制動時に回収されたエネルギー量

## トルクベース vs パワーベースのドライバー要求

:::info バージョン 2025.2 以降の新機能
CarSim 2025.2 以降では、ドライバーモデルからのパワートレインへの要求信号として、従来の**トルクベース**に加えて**パワーベース**の指定が可能になりました。

- **トルクベース** — アクセルペダル開度 → 要求トルクに変換。モーターの応答特性を直接評価する場合に適切
- **パワーベース** — アクセルペダル開度 → 要求パワーに変換。エネルギー管理戦略の評価に適切

テスト目的に応じて適切な方式を選択してください。
:::

## カスタム EV テストの作成手順

1. **アセンブリタイプの選択** — 車両アーキテクチャに合った電動パワートレインを選択
2. **モーター・バッテリーパラメータの設定** — 評価対象の仕様値を入力
3. **走行シナリオの定義** — Procedure で速度パターンや走行サイクルを設定
4. **PMC の調整** — 回生配分や出力制限を制御戦略に合わせて調整
5. **結果の比較** — 異なるパラメータ設定での結果を Overlay で比較

:::note
EV テストでは、バッテリーの初期 SOC 設定がシミュレーション結果に大きく影響します。比較テストを行う場合は、初期 SOC を統一して条件を揃えることが重要です。
:::
