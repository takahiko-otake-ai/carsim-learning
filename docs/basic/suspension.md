# サスペンション

CarSimのサスペンションモデルは、キネマティクス（幾何学的運動特性）とコンプライアンス（弾性変形特性）の両面から車両の懸架装置を再現します。

## サスペンションタイプ

Vehicle Assembly画面で選択可能なサスペンションタイプは以下の通りです。

| タイプ | 適用 | 説明 |
|---|---|---|
| **Generic/Independent** | 前後 | 独立懸架、トーションビーム用の汎用モデル |
| **Solid Axle** | 前後 | リジッドアクスル（車軸式） |
| **Virtual Steering Axis (VSA)** | 前のみ | 仮想ステアリング軸を使用する独立懸架 |
| **Solid Axle with Wheel Compliance** | 前後 | ホイールコンプライアンス付きリジッドアクスル |
| **Twist Beam 2016/Legacy** | 後のみ | トーションビーム専用モデル |
| **No Rear Suspension** | 後のみ | リアサスなし（連結車両用） |

:::info Twist Beamモデルの改良（2018.0+）
バージョン2018.0以降、Twist Beamモデルが大幅に改良されました。ビームのねじり剛性とロール特性のカップリングがより正確にモデル化されています。
:::

## キネマティクス

サスペンションキネマティクスは、**バウンス（ジャウンス）ベースのテーブルマップ**で定義されます。サスペンションストローク量に対する以下の変化をテーブルで入力します。

- **ホイールセンター軌跡**: ストロークに伴うホイール中心の移動
- **トー変化**: ストロークに伴うトー角の変化
- **キャンバー変化**: ストロークに伴うキャンバー角の変化
- **キャスター変化**: ストロークに伴うキャスター角の変化
- **横方向変位**: ホイールの車幅方向の移動
- **前後方向変位**: ホイールの車両前後方向の移動

### 独立懸架とソリッドアクスルのキネマティクスの違い

- **独立懸架**: 各車輪が独立してストロークし、それぞれのキネマティクス特性に従います
- **ソリッドアクスル**: 左右の車輪がアクスルで結合されているため、片輪のストロークが反対側にも影響します。ロールセンター位置やアクスルのステア特性も考慮されます

## コンプライアンス

サスペンションのコンプライアンス特性は以下の要素で構成されます。

### スプリング

スプリングモデルは伸び側と圧縮側のそれぞれに特性テーブルを設定できます。

- **伸び側テーブル（Extension）**: リバウンド方向の荷重-変位特性
- **圧縮側テーブル（Compression）**: バンプ方向の荷重-変位特性
- **β値**: スプリングのモーションレシオ（レバー比）を定義するパラメータ

:::note β値について
β値は、ホイールストロークに対するスプリング変位の比率を表します。β = 1.0 の場合、ホイールストロークとスプリング変位が同一であることを意味します。
:::

### ダンパー

ダンパー（ショックアブソーバー）は速度依存の減衰力特性をテーブルで定義します。伸び側と圧縮側で異なる特性を設定可能です。

### バンプストップ

サスペンションのフルバンプ・フルリバウンド時の衝撃を吸収するバンプストップの特性を設定します。

### ロール剛性

ロール剛性（スタビライザーを含む）の入力方法は2種類あります。

- **直接入力**: ロール剛性値を直接指定
- **導出**: スプリング特性とジオメトリから自動計算

### ブッシュコンプライアンス

サスペンションブッシュ（ゴムブッシュ）の弾性変形を考慮します。横力やブレーキ力によるサスペンションの変形を再現し、操縦安定性やブレーキ時の挙動に影響します。

:::tip 設計への活用
サスペンションコンプライアンスの設定を変更することで、アンダーステア/オーバーステア特性のチューニング、制動時のノーズダイブ量、加速時のスクォート量などを評価できます。
:::
