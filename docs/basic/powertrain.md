# パワートレイン

CarSimのパワートレインモデルは、従来の内燃機関（ICE）から電気自動車（EV）、ハイブリッド車（HEV）まで、幅広い駆動構成に対応しています。

## 駆動方式

CarSimは以下の駆動方式をサポートしています。

| 駆動方式 | 略称 | 説明 |
|---|---|---|
| 前輪駆動 | FWD | フロントアクスルのみに駆動力を配分 |
| 後輪駆動 | RWD | リアアクスルのみに駆動力を配分 |
| 全輪駆動 | AWD | 前後アクスルに駆動力を配分 |

## 動力源

- **ICE（内燃機関）**: エンジントルク特性マップに基づくモデル
- **EV（電気自動車）**: モーター/ジェネレータの特性とバッテリー管理
- **HEV（ハイブリッド）**: ICEとモーターの併用

## パワートレイン構成要素

パワートレインは以下のコンポーネントで構成されます。

- エンジン
- モーター/ジェネレータ
- トルクコンバータ／クラッチ
- トランスミッション
- プラネタリギア
- センターデフ
- デファレンシャル（左右）

### デファレンシャルの種類

| 種類 | 特徴 |
|---|---|
| **ビスカスカップリング** | 回転速度差に応じてトルクを配分 |
| **トルクセンシングLSD** | 入力トルクに応じたロック率 |
| **トルクベクタリング** | 能動的に左右トルク配分を制御 |

## HEV/EVアセンブリタイプ

CarSimは4種類のHEV/EVアセンブリタイプを用意しています。

### 1. Power Split（パワースプリット）
プラネタリギアを用いてエンジンとモーターの動力を分配する構成です（例：Toyota THS）。

### 2. Parallel Hybrid（パラレルハイブリッド）
エンジンとモーターが同じ駆動軸上に並列接続される構成です。

### 3. Series Hybrid（シリーズハイブリッド）
エンジンは発電専用で、モーターのみが駆動力を供給します。

### 4. Electric（電動）
バッテリーとモーターのみで構成されるピュアEVです。

## モーター配置の柔軟性

CarSimでは**各アクスルに0〜2基**のモーターを配置可能です。インホイールモーターにも対応しており、独立した左右モーター制御が行えます。

:::info モーター配置の例
- フロント1基 + リア1基：標準的なAWD EV
- 各輪1基（計4基）：インホイールモーター構成
- リア2基：リアアクスルのトルクベクタリング
:::

## バッテリー管理

EVおよびHEVモデルでは、以下のバッテリーパラメータを設定できます。

- **容量**（kWh）
- **内部抵抗**
- **電圧特性**（SOCに応じた電圧変化）
- **温度特性**

## EV設定画面

EV関連の設定は以下の2つの画面で行います。

- **Power Management Control（PMC）**: エネルギー管理戦略、回生ブレーキ制御
- **Hybrid/Electric Assembly**: モーター配置、バッテリー構成、パワートレインレイアウト

:::tip ドライバーデマンド（2025.2+）
バージョン2025.2以降では、ドライバーのアクセルペダル要求を**トルクベース**と**パワーベース**の2種類から選択できます。パワーベースでは、速度に応じたより自然なドライバー操作が再現されます。
:::

## パワートレインマウント

パワートレインマウントは、パワートレイン反力が車体に伝達される際の特性をモデル化します。マウントの剛性・減衰特性を設定することで、急加速・急減速時のパワートレイン挙動をより正確に再現できます。
