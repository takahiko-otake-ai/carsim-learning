# 空力

CarSimの空力モデルは、車両に作用する空気力学的な力とモーメントを計算します。

## 相対風速・風向の計算

空力の計算には、車両速度、車両の向き（ヘディング角）、および設定された風の条件から**相対風速**と**相対風向**が算出されます。

:::note
空力計算ではアニメーション用の車両形状は考慮されません。空力特性は係数マップによって独立に定義されます。
:::

## 6自由度の空力係数

CarSimでは**6自由度**（3方向の力 + 3軸まわりのモーメント）の空力係数を設定します。各係数は**空力スリップ角**（相対風向角）に対するマップとして入力可能です。

| 係数 | 説明 |
|---|---|
| **Cx** | 前後方向力係数（ドラッグ） |
| **Cy** | 横方向力係数（サイドフォース） |
| **Cz** | 垂直方向力係数（リフト/ダウンフォース） |
| **Cmx** | ロールモーメント係数 |
| **Cmy** | ピッチモーメント係数 |
| **Cmz** | ヨーモーメント係数 |

## 空力計算式

空力の各成分は以下の基本式で計算されます。

`F = (1/2) × ρ × V² × A × C`

ここで：

- **ρ**: 空気密度
- **V**: 相対風速
- **A**: 前面投影面積
- **C**: 該当する空力係数

モーメント成分には前面投影面積に代わり、**縦方向基準長**（通常はホイールベース）が使用されます。

## 主要パラメータ

空力モデルの設定には以下のパラメータが必要です。

- **基準点座標**: 空力の作用点（車両座標系での位置）
- **前面投影面積**: 車両の正面から見た投影面積（m²）
- **縦方向基準長**: モーメント計算の基準長さ（m）
- **空気密度**: 大気の密度（kg/m³）

:::warning 基準点の設定
空力係数の基準点は、風洞試験やCFD解析の基準点と一致させる必要があります。基準点が異なる場合、モーメント係数の変換が必要です。
:::

:::tip 詳細情報
空力モデルの詳細については、CarSimアプリケーション内の **Help > Aerodynamics** を参照してください。
:::
