# タイヤモデル

タイヤはCarSimのシミュレーションにおいて最も重要なコンポーネントの一つです。タイヤモデルの役割は、**接地点の決定**と**力・モーメントの計算**の2つに大別されます。

## タイヤの力とモーメント

タイヤは接地面（コンタクトパッチ）において以下の6成分の力・モーメントを発生します。

| 成分 | 記号 | 説明 |
|---|---|---|
| 前後力 | Fx | 駆動力・制動力（タイヤ回転方向） |
| 横力 | Fy | コーナリングフォース（タイヤ横方向） |
| 垂直力 | Fz | 接地荷重 |
| オーバーターニングモーメント | Mx | キャンバースラストによるモーメント |
| 転がり抵抗モーメント | My | 転がり抵抗に起因するモーメント |
| セルフアライニングトルク | Mz | ニューマチックトレイルによるモーメント |

## 基本概念

### スリップ角

スリップ角は、タイヤの進行方向とタイヤの向きの間の角度差です。横力（コーナリングフォース）はスリップ角に応じて発生します。

### キャンバー角

キャンバー角は、タイヤの傾きを表す角度です。キャンバー角によりキャンバースラスト（横力成分）が発生します。

## 内蔵タイヤモデル

### テーブルモデル

CarSimの標準的なタイヤモデルで、以下の特性をマップ（テーブル）として入力します。

- **Fx**: スリップ比と垂直荷重に対する前後力
- **Fy**: スリップ角と垂直荷重に対する横力
- **Mz**: スリップ角と垂直荷重に対するセルフアライニングトルク
- **キャンバースラスト**: キャンバー角に対する横力成分

### Pacejka 5.2

Magic Formula（Pacejka 5.2）モデルに対応しています（2021.0まで）。係数セットによりタイヤ特性を数式で表現します。

:::note Pacejkaモデルについて
Pacejka 5.2はCarSim 2021.0までのバージョンで内蔵モデルとして利用可能です。それ以降のバージョンでは、外部タイヤモデルインターフェース（VS STI）経由での利用が推奨されます。
:::

## 外部タイヤモデル（VS STI）

VS STI（Standard Tire Interface）を通じて、以下の外部タイヤモデルと連携できます。

| モデル名 | 提供元 | 対応バージョン |
|---|---|---|
| **Telemecanics** | — | 2023.0+ |
| **MF-Tyre/MF-Swift** | Siemens | — |
| **FTire** | COSIN | — |
| **TameTire** | Michelin | — |
| **CDTire** | Fraunhofer | 2024.2+ |
| **カスタムモデル** | ユーザー定義 | — |

:::info MF-Tyre/Swiftインターフェースの改良（2025.1+）
バージョン2025.1以降、MF-Tyre/Swiftインターフェースが改良され、タイヤの更新周波数を設定可能になりました（`DELFT_TYRE_FREQ`パラメータ）。高周波のタイヤ応答が必要なシミュレーションで計算精度と速度のバランスを調整できます。
:::

## フリクションサークル

タイヤが発生可能な前後力と横力の組み合わせには限界があり、この限界を表すのが**フリクションサークル**（摩擦円）の概念です。CarSimでは**Bakkerのコンバインドスリップ理論**に基づいてこの限界を計算します。

## 摩擦係数補正

路面摩擦係数の補正方法は2種類あります。

- **単純スケーリング**: 力・モーメントに一定のスケーリング係数を乗算
- **類似法（Similarity Method）**: Pacejkaの類似法に基づく補正。摩擦係数の変化に対してタイヤ特性の形状を維持しながらスケーリング

## 過渡特性

### 緩和長（リラクゼーションレングス）

タイヤの横力発生には遅れがあり、この遅れを表すパラメータが**緩和長**です。ステアリング操作に対するタイヤ応答の遅れを再現します。

### ターンスリップ効果（2025.2+）

バージョン2025.2以降、**ターンスリップ効果**が導入されました。低速域・据え切り時のステアリングフィードバックが改善され、駐車場面での操舵トルクがより現実的に再現されます。

:::tip タイヤモデルの選択指針
- 基本的な車両運動解析にはテーブルモデルで十分です
- タイヤの詳細な非線形特性が必要な場合はMF-Tyre等の外部モデルを使用してください
- 高周波振動やタイヤのベルト挙動を再現する場合はFTireやMF-Swiftが適しています
:::
