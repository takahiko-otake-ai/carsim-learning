# ステアリング

CarSimのステアリングモデルは、ドライバーの操舵入力から各車輪の転舵角が決定されるまでの一連のメカニズムを再現します。

## 入力タイプ

ステアリングの入力は以下の2種類から選択できます。

- **角度入力**: ステアリングホイールの回転角を直接指定
- **トルク入力**: ステアリングコラムに加わるトルクを指定

:::info 入力タイプの選択
角度入力はオープンループ試験（定常円旋回、レーンチェンジなど）に適しています。トルク入力はドライバーモデルとの連携や、ステアリングフィールの評価に使用されます。
:::

## ギアタイプ

CarSimは以下のステアリングギアタイプに対応しています。

| ギアタイプ | 特徴 |
|---|---|
| **ラック＆ピニオン** | 乗用車で主流。ピニオンの回転をラックの直線運動に変換 |
| **リサーキュレーティングボール** | トラック・大型車両で使用。ボールねじ機構による変換 |

## ステアリングキネマティクス（アッカーマン）

ステアリングキネマティクスでは、ステアリングホイール角から左右の前輪転舵角への変換を定義します。**アッカーマンジオメトリ**を考慮することで、旋回時の内輪・外輪の転舵角差を再現します。

## ステアリングコンプライアンス

ステアリング系の弾性変形をモデル化します。コンプライアンスにより、路面からの入力やコーナリング力によるステアリング系のたわみが再現されます。

## キングピンジオメトリ

キングピン軸の定義方法は2種類あります。

### 固定軸（Fixed Axis）
キングピン軸を固定の傾斜角とオフセットで定義する従来の方式です。

### 仮想ステアリング軸（Virtual Steering Axis: VSA）
サスペンションストロークに応じてキングピン軸が変化するモデルです。マルチリンクサスペンションなど、実車のジオメトリをより忠実に再現できます。

## パワーアシスト

パワーステアリングのアシスト力は以下の2箇所で設定可能です。

- **コラムアシスト**: ステアリングコラム部でトルクをアシスト（電動パワステ向け）
- **ラック/ギアアシスト**: ラック部またはギア部でアシスト（油圧パワステ向け）

## 転舵角を構成する要素

最終的な車輪の転舵角は、以下の複数の要素の合成として決定されます。

1. **ステアリングキネマティクス**: ステアリングホイール角からの変換
2. **ステアリングコンプライアンス**: ステアリング系の弾性変形
3. **キングピンモーメント**: キングピン軸まわりの力による変位
4. **サスペンションキネマティクス**: サスペンションストロークに伴うトー変化
5. **サスペンションコンプライアンス**: サスペンションブッシュ等の弾性変形

:::note
これらの要素が複合的に作用することで、実車における複雑な操舵特性（セルフアライニングトルク、キックバック等）が再現されます。
:::

## Steering画面の構成

Steering設定画面には以下の項目が含まれています。

- コラム特性（慣性モーメント、減衰）
- キングピンジオメトリ
- ステアリングコンプライアンス
- 摩擦・減衰特性
- ギアタイプ・ギア比
- スタティックトルク
- パワーアシスト特性
- ステアリングキネマティクス
- ステアリングストッパー（舵角制限）
