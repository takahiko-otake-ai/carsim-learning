# ブレーキシステム

CarSimのブレーキモデルは、ペダル入力からブレーキトルク発生までの一連のプロセスをシミュレーションします。

## ブレーキ入力タイプ

ブレーキへの入力方法は以下の3種類から選択できます。

| 入力タイプ | 説明 |
|---|---|
| **マスターシリンダ圧力** | 直接圧力値を入力 |
| **ペダル踏力（ブースターあり）** | ブースター特性を考慮したペダル踏力入力 |
| **ペダル踏力（ブースターなし）** | ブースターなしのペダル踏力入力 |

## プロポーショニングバルブ

前後ブレーキ圧力の配分を制御するプロポーショニングバルブには、以下の設定方法があります。

- **係数入力**: 前後の圧力比を一定の係数で定義
- **テーブル入力**: マスターシリンダ圧力に対する前後圧力の関係をテーブルで定義

:::tip タイヤ垂直荷重の考慮
プロポーショニングバルブの設定では、タイヤの垂直荷重（法線力）を考慮した配分も可能です。制動時の荷重移動に応じた動的な前後配分が再現できます。
:::

## ABSモデル

CarSimには簡易的なABSモデルが組み込まれています。

- **制御方式**: スリップ比制御
- **使用**: オプション（有効/無効を選択可能）

:::note
より高度なABS制御ロジックが必要な場合は、Simulink等の外部コントローラとの連携が推奨されます。CarSimの内蔵ABSモデルは基本的な動作確認用として設計されています。
:::

## 油圧遅延オプション

ブレーキペダル操作からブレーキ圧力発生までの遅延を再現するために、3種類のモデルが用意されています。

| 遅延モデル | 説明 |
|---|---|
| **一次遅れ** | 時定数で定義されるシンプルな遅延モデル |
| **アクチュエータ流体体積** | 油圧システムの流体体積に基づく物理的な遅延モデル |
| **遅延なし** | 即座にブレーキ圧力が発生 |

## ブレーキトルクの計算

ブレーキトルクは以下の方法で計算されます。

- **係数入力**: ブレーキ圧力に対して一定の係数でブレーキトルクを算出
- **テーブル入力**: ブレーキ圧力とブレーキトルクの関係をテーブルで定義
- **温度特性**: ブレーキ温度に応じたトルク変化（フェード特性）の考慮が可能

## 画面構成：Brakes: Four-Wheel System

ブレーキ関連のパラメータは「Brakes: Four-Wheel System」画面で一元的に管理されます。この画面から入力タイプの選択、プロポーショニングバルブの設定、各輪のブレーキ特性の定義が行えます。

## ベアリング摩擦

各車輪のベアリング摩擦トルクを設定することで、コースティング時や極低速域での車両挙動をより正確に再現できます。ベアリング摩擦は車輪回転に対する抵抗として常に作用します。

:::warning 注意事項
ブレーキトルクの温度特性を使用する場合、ブレーキディスク/ドラムの熱容量パラメータも適切に設定してください。不適切な設定はフェード特性の再現精度に影響します。
:::
