# 道路と路面

CarSimでは、車両が走行する道路環境を詳細に定義できます。パス（走行軌跡）の設定から路面の高低差、摩擦係数、アニメーション表示まで、多彩な機能が用意されています。

## パス（走行軌跡）

パスはX-Y平面上に定義される走行経路で、**Path ID** により管理されます。

### S-L座標系

CarSimではパスに沿った座標系として **S-L座標系** を使用します。

| 座標軸 | 名称 | 説明 |
|-------|------|------|
| S | Station | パスに沿った距離（走行方向） |
| L | Lateral | パスに垂直な距離（左方向が正） |

:::info
X-Y座標からS-L座標への変換は反復計算を用いるため、パス形状によっては**収束しない場合**があります。急激なカーブや交差するパスでは注意が必要です。
:::

### パス定義方法

- **X-Y座標**: 平面上の座標列で定義
- **X-Y-Z座標**: 3次元座標列で定義（高低差を含む）
- **GPS変換**: GPS座標データからパスを生成
- **Segment Builder**: 直線区間と曲線区間を組み合わせて構築

#### Segment Builderの設定項目

Segment Builderでは以下の要素を指定してパスを構築します。

- パス開始点（X, Y座標）
- 開始角度
- Station座標
- 直線セグメントと曲線セグメント
- 閉じたパス（周回路）の設定
- GPS原点

## 路面高さ（Elevation）

路面の高低差は3つの方法で定義できます。

| 方法 | 説明 | 用途例 |
|------|------|-------|
| Reference Line Elevation | S座標に対するZ値 | 坂道、丘陵路 |
| Off-Center Elevation S-L Grid | S-Lグリッド上のdZ値 | バンプ、カント |
| Off-Center Elevation Variable Width | 端部基準のdZ値 | 轍、合流・分岐、車線変更 |

## 摩擦係数

路面の摩擦特性を定義します。

- **路面摩擦係数**: 定数値またはテーブル（S座標やL座標に応じた可変設定）
- **転がり抵抗**: タイヤと路面間の転がり抵抗係数

## アニメーション表示

道路のビジュアル表現に関する設定です。

- **路面テクスチャ**: RGB/Color、Material Texture Image、Spec、Detail
- **静的オブジェクト**: 建物や標識などの配置（センサ検知対象外）
- **空・背景**: スカイボックスの設定

## Road画面

3D Surfaceの画面レイアウトで道路を構成します。各道路には**Road ID**が割り当てられ、複数の道路を管理できます。

## VS Terrain

より詳細な走行環境をモデリングするための機能です。

- **GPU並列演算**による高速処理
- 複数のテレイン要素の**マージ（統合）**
- **境界条件**の設定

### VS Terrain Utility

FBXファイルを `.vsterrain` 形式に変換するユーティリティです。Unreal Engineとの統合に活用できます。

### VS Scene Builder

- タイルベースの道路構築
- **OpenDRIVE（.xodr）** 形式を `.vsscene` 形式に変換
- 車両やオブジェクトのパス定義

:::tip
**ATLAS GPSデータ**（atlas.carsim.com）を利用すると、GoogleやHEREの地図データからパスを設定できます。ただし、車線幅や路面高さの情報は含まれないため、別途設定が必要です。
:::
