# 結果の可視化

CarSimのシミュレーション結果は **VS Visualizer** を使って3Dアニメーションやグラフで確認できます。直感的な操作で車両挙動を多角的に分析できます。

## VS Visualizerの操作

### アニメーション操作

3Dアニメーション画面では、マウスとキーボードの組み合わせでカメラを自在に操作できます。

| 操作 | 方法 |
|------|------|
| カメラ位置 | マウスドラッグ |
| ズーム | Ctrl + Shift + マウス |
| カメラロール | Shift + マウス |
| 視点移動 | Ctrl + マウス |
| ドリーズーム | ズーム＋視野角の同時変更 |

### グラフ操作

プロット画面では豊富なショートカットキーで効率的にデータを確認できます。

| 操作 | キー/方法 |
|------|----------|
| コンテキストメニュー | 右クリック |
| グラフ最大化 | Z キー、またはダブルクリック |
| グラフ選択 | Fn + Up / Down |
| ズームイン/アウト | マウスホイール |
| 水平方向ズーム | A / D キー、またはホイール横ドラッグ |
| 垂直方向ズーム | X / W キー、またはホイール縦ドラッグ |
| 部分ズーム | 左クリックドラッグ |
| リセット | R キー |
| 数値表示 | V キー |
| グリッド切替 | G キー |
| 再生/一時停止 | Space キー |

:::tip
**Z キー**でグラフを最大化すると、詳細なデータ確認が容易になります。**V キー**で数値を表示すれば、特定時刻のデータ値を正確に読み取れます。
:::

## 確認可能なファイル

VS Visualizerから以下のファイルを閲覧できます。

| ファイル | 内容 |
|---------|------|
| Echoファイル | 初期条件と最終条件を含むパラメータ記録 |
| 全データファイル | 数学モデルとビジュアライザの全データ |
| Logファイル | Parsfileとイベントの実行記録 |
| シミュレーション結果 | CSV、MAT形式での出力データ |
| VS/ERDヘッダファイル | 出力変数の定義情報 |
| 状態変数ファイル | 状態変数の時系列データ |
| Importsファイル | 外部入力データ |
| Outportsファイル | 外部出力データ |

## Plot Setup画面

カスタムプロットを作成するための設定画面です。

- 出力ファイルから表示する**信号の選択**
- **凡例**・**ラベル**・**色**・**線種**のカスタマイズ
- 新しい変数の定義とプロット
- 演算式を用いた派生変数の作成

## 結果の管理と比較

:::info
複数のシミュレーション結果を同一グラフ上に重ねて比較できます。車両パラメータの変更やドライバーモデルの違いによる挙動差を視覚的に分析する際に有用です。
:::

### 出力ファイル管理

- シミュレーションごとに結果ファイルが自動保存されます
- ファイル名やディレクトリでランを整理・管理できます
- 過去の結果を再読み込みしてプロットに追加可能です
- CSV/MAT形式でエクスポートし、外部ツール（MATLAB、Excelなど）で詳細分析も行えます
