# ドライバーモデル

CarSimでは、車両の操作入力を**オープンループ制御**と**クローズドループ制御**の2つのアプローチで定義できます。シミュレーションの目的に応じて適切な制御モデルを選択することが重要です。

## オープンループ制御

オープンループ制御では、ドライバーの操作入力を時系列データとして直接指定します。フィードバックなしに、あらかじめ定義された操作が再生されます。

| 操作項目 | 入力パラメータ |
|---------|-------------|
| ブレーキ | マスターシリンダー圧力、またはペダル踏力 |
| アクセル | スロットル開度 |
| シフト | ギア段、クラッチ操作 |
| ステアリング | ハンドル角度、またはトルク |

:::tip
オープンループ制御は再現性が高く、車両特性の比較試験に適しています。同一の入力で異なる車両設定を評価できます。
:::

## クローズドループ制御

### 速度制御

クローズドループ速度制御には複数のモードがあり、`OPT_SC` パラメータで切り替えます。

| OPT_SC | モード | 説明 |
|--------|------|------|
| 1, 2, 3 | Speed Control | 目標速度に追従する基本的な速度制御 |
| 4 | Path Preview | 前方のカーブ曲率を参照し、最大パフォーマンスを発揮する高度な制御 |
| 5 | Acceleration Control | 加速度コマンドによる制御 |

**Path Preview（OPT_SC=4）** は最も高度な速度制御モードです。前方道路の曲率情報を参照し、以下のパラメータで挙動を調整します。

- **アグレッシブネス（攻めの度合い）**: ドライバーがどの程度限界走行を行うかを設定
- **最大G設定**: 前後方向・横方向の加速度上限を個別に設定
- **ドライバースキル（摩擦円）**: フリクションサークルの活用度でドライバーの技量を表現

### ステアリング制御

ステアリングのクローズドループ制御には2つのモデルがあります。

#### シングルポイントプレビューモデル（デフォルト）

車両前方の**プレビューポイント**と**目標パス**との横方向偏差を検出し、その偏差を最小化するようにステアリングを操作します。シンプルで安定した制御を実現します。

#### MacAdamモデル（10点プレビュー）

10個のプレビューポイントを使用する高度なモデルです。車速とプレビュー時間から算出されるプレビュー距離にわたって、予測経路と目標経路の誤差を最小化します。

`プレビュー距離 = 車速 × プレビュー時間`

より人間に近い先読み運転を再現できるため、限界走行や高速走行のシミュレーションに適しています。

:::note
オープンループとクローズドループを切り替えるには、**ライブラリの変更**が必要です。同一ライブラリ内でのモード変更はできないため、事前にシミュレーション条件を確認してください。
:::

## 制御モデル選択のガイドライン

- **車両特性の比較評価** → オープンループ（同一入力での再現性重視）
- **目標経路への追従シミュレーション** → クローズドループ・ステアリング
- **限界走行・レーストラック走行** → Path Preview（OPT_SC=4）＋ MacAdamモデル
- **ADAS制御アルゴリズムの検証** → 外部コントローラ連携（加速度制御 OPT_SC=5）
