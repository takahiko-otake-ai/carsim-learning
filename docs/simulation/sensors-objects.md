# センサとオブジェクト

CarSimでは、ADAS（先進運転支援システム）やAD（自動運転）のシミュレーションに不可欠な**ムービングオブジェクト**と**各種センサ**を詳細に設定できます。

## ムービングオブジェクト

ムービングオブジェクトは、シミュレーション環境における周囲の車両、歩行者、標識などを表現します。ADASセンサの検知対象となります。

- **最大数**: 200個まで配置可能

### 設定項目

| 項目 | 説明 |
|------|------|
| 寸法 | センサ検知範囲に影響するオブジェクトサイズ |
| 参照パス/道路 | オブジェクトが走行する経路 |
| S-L移動情報 | 位置、速度、加速度の時系列データ |
| アニメーション | 3D表示の外観設定 |

:::info
ムービングオブジェクトは**外部制御**にも対応しています。Simulinkや外部プログラムからリアルタイムに位置・速度を制御できます。
:::

### 設定画面

- **Multiple Moving Objects**: 複数オブジェクトの一括管理画面
- **Single Moving Object (Custom)**: 個別オブジェクトの詳細設定画面

## ADASセンサ

ADASセンサは車両に搭載され、ムービングオブジェクトとの相対情報を取得します。

- **最大数**: 99個まで搭載可能

### 取得可能な情報

- 対象物との**距離**
- 相対**速度**
- 相対**角度**

### 設定項目

| 項目 | 説明 |
|------|------|
| 取り付け位置 | 車両上のセンサ搭載座標 |
| 取り付け角度 | センサの向き |
| 検知範囲 | 距離・角度の検知限界 |
| 感度 | センサの検知感度 |
| センサオクルージョン | 他オブジェクトによる遮蔽の考慮 |
| アニメーション | センサ検知範囲の3D表示 |

:::note
センサの角度は**外部制御**が可能です。スキャン動作を模擬するLIDARや、追跡モードのレーダーなどを再現できます。
:::

### 設定画面

**ADAS Sensors for Range and Tracking** 画面で、検知範囲と追跡設定を一括管理します。

## モーションセンサ

モーションセンサは、車両のバネ上/バネ下ボディの任意の位置・姿勢に取り付けられる計測センサです。

### 出力データ

**絶対座標系（グローバル）**:
- 位置
- 速度
- 加速度

**局所座標系（ローカル）**:
- 速度
- 加速度
- ジャーク（加加速度）
- 角速度

さらに、**求心加速度**と**接線加速度**も出力可能です。

### 設定画面

**Positions, Velocities, and Accelerations** 画面で、センサ位置と出力変数を設定します。

:::tip
モーションセンサはIMU（慣性計測装置）のシミュレーションに最適です。任意の位置に設置して、実車のセンサ取り付け位置に合わせた出力を得ることができます。
:::
