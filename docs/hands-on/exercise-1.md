# 演習1: CarSim の起動と最初の実行

## 学習目標

- CarSim GUI の起動方法とデータベースの選択を理解する
- ライセンス設定の手順を習得する
- データセットの複製（Duplicate）操作を実行できるようになる
- シミュレーションの実行とビデオ・プロットの確認方法を学ぶ

## 前提条件

- CarSim がインストール済みであること
- 有効なライセンス（ドングルまたはネットワークライセンス）が利用可能であること
- サンプルデータベースがインストールされていること

---

## 手順

### ステップ 1: CarSim の起動

Windows のスタートメニューまたはデスクトップショートカットから CarSim を起動します。

:::tip
デスクトップに CarSim のショートカットが見つからない場合は、スタートメニューで「CarSim」と検索してください。
:::

### ステップ 2: データベースの選択

**Select Recent Database** ウィンドウが表示されます。使用するデータベースを選択し、**Open the Selected Database** をクリックします。

初回起動時はデフォルトのサンプルデータベースを選択してください。

### ステップ 3: ライセンス設定

**License Settings** 画面が表示されたら、必要な **Feature** にチェックを入れ、**Select** ボタンをクリックします。

:::info
ドングルライセンスを使用する場合は、**"Scan for Dongle License"** オプションにチェックを入れてください。ネットワークライセンスの場合は、ライセンスサーバーのアドレスが正しく設定されていることを確認します。
:::

### ステップ 4: データセットの複製

1. **Datasets** タブに移動します
2. ツリーから **&#123;Powertrain: ICE&#125; 8-Speed Trans, Var. Shift Duration** を見つけます
3. **Duplicate** ボタンをクリックします
4. 新しいデータセットの情報を入力します:
   - **Category**: `** CS Training`
   - **Title**: `#1 C-Class, Acceleration`

:::caution
Category の先頭に `**` を付けることで、ツリーの上部にソートされ、トレーニング用データセットを見つけやすくなります。
:::

### ステップ 5: 車両のリンク

Vehicle のリンクを **&#123;C-Class&#125; C-Class, Hatchback** に設定します。ドロップダウンまたは Browse ボタンから選択できます。

### ステップ 6: 車両カラーの設定

画面右上のカラー設定で車両の色を**赤（Red）**に変更します。シミュレーション結果のビデオで車両の色が反映されます。

### ステップ 7: カメラの設定

Camera setup を以下のように変更します:

| パラメータ | 値 |
|---|---|
| プリセット | **&#123;Vehicle Reference&#125;** |
| Azimuth | **150°** |
| Elevation | **10°** |
| Distance | **6.25 m** |

### ステップ 8: オーバーレイの設定

**"Overlay videos and plots with other runs"** チェックボックスをオンにします。この機能を使うと、複数のシミュレーション結果を重ねて比較できます。

### ステップ 9: オーバーレイ対象の指定

オーバーレイの対象として、複製元のオリジナルデータセットをリンクします。これにより、パラメータ変更前後の結果を視覚的に比較できるようになります。

### ステップ 10: シミュレーションの実行

**Run Math Model** ボタンをクリックし、表示されるダイアログで **Execute** を選択してシミュレーションを実行します。

:::warning
シミュレーション実行中は CarSim を閉じないでください。計算が完了するまで待ちます（通常は数秒〜数十秒）。
:::

### ステップ 11: 結果の確認

1. **Video + Plot** ボタンをクリックして結果を表示します
2. ビデオウィンドウで車両の走行をアニメーションで確認します
3. プロットウィンドウで速度、加速度などの時系列データを確認します
4. **VS Visualizer** を開き、3D ビューで結果を詳しく確認します

:::tip
VS Visualizer では、マウスのドラッグで視点を回転、スクロールでズームイン・ズームアウトが可能です。再生速度の変更やフレームごとのステップ再生も試してみましょう。
:::

---

## 期待される結果

- CarSim GUI が正常に起動し、データベースが読み込まれること
- 複製したデータセットが `** CS Training` カテゴリに表示されること
- シミュレーションが正常に完了し、加速走行のビデオとプロットが表示されること
- オーバーレイ機能により、オリジナルと複製の結果が重ねて表示されること
