# 演習2: Copy and Link とライブラリ切り替え

## 学習目標

- **Copy and Link** 操作の概念と使い方を理解する
- Procedure（手順）データセットの編集方法を習得する
- ライブラリの切り替えによるモデル構成の変更方法を学ぶ
- オープンループとクローズドループのステアリング制御の違いを理解する

## 前提条件

- [演習1](./exercise-1.md) を完了していること
- `** CS Training` カテゴリに `#1 C-Class, Acceleration` が作成済みであること

---

## Part 1: Copy and Link

### ステップ 1: データセットの複製

`#1 C-Class, Acceleration` を選択し、**Duplicate** を実行します。

- **Category**: `** CS Training`
- **Title**: `#2 C-Class, 60 km/h`

### ステップ 2: Procedure の Copy and Link

Procedures セクションに表示されている **"Ramp to 0.5 Throttle"** を右クリックし、コンテキストメニューから **Copy and Link** を選択します。

新しい名前を入力します: `Constant 60 km/h`

:::info
**Copy and Link** は、元のデータセットのコピーを作成し、そのコピーを現在のデータセットにリンクする操作です。元のデータセットは変更されないため、他のシミュレーションに影響を与えません。**Duplicate** との違いは、Copy and Link ではリンクが自動的に新しいコピーに更新される点です。
:::

### ステップ 3: Procedure の編集

青色のリンク **"Constant 60 km/h"** をクリックして、Procedure の編集画面に移動します。

### ステップ 4: ドライバー制御の変更

1. **Driver Controls** タブを開きます
2. コントロールモードを **"Throttle + initial speed"** から **"Constant target speed"** に変更します
3. 目標速度を **60 km/h** に設定します

### ステップ 5: プロットの設定

1. Home 画面に戻ります
2. **More plots** の数を **5** に設定します
3. 速度プロットをリンクして、シミュレーション結果で速度の時間変化を確認できるようにします

### ステップ 6: 実行と確認

シミュレーションを実行し、車両が 60 km/h の定速走行を行うことを確認します。

:::tip
プロットで速度が 60 km/h 付近で安定していれば、設定は正しく適用されています。
:::

## Part 2: ライブラリ切り替え

### ステップ 1: データセットの複製

`#2 C-Class, 60 km/h` を選択し、**Duplicate** を実行します。

- **Category**: `** CS Training`
- **Title**: `#3 C-Class, 60 km/h, Open-Loop Steering`

:::tip
タイトルを間違えた場合は、**Edit → Change Title or Category** から修正できます。
:::

### ステップ 2: Procedure の Copy and Link

Procedure を **Copy and Link** し、新しい名前を設定します: `Constant 60 km/h, 90 deg Steering`

### ステップ 3: ステアリングライブラリの確認

Procedure 内の Steering セクションを開き、現在使用されているライブラリが **Driver Path Follower** であることを確認します。

Driver Path Follower は、指定された走行経路に沿って自動的にステアリングを制御するクローズドループ制御です。ドライバーモデルが経路からの逸脱を検知し、ステアリング角を調整します。

### ステップ 4: ライブラリの切り替え

ステアリングライブラリを **Driver Path Follower** から **Control: Steering (Open Loop)** に切り替えます。

ライブラリの切り替えはドロップダウンメニューまたは Browse ボタンから行えます。

### ステップ 5: ステアリング入力の設定

**Ramp: 0 to 90 deg. in 0.25 sec** を選択します。

これにより、0.25 秒間でステアリング角が 0° から 90° まで直線的に変化するオープンループ入力が設定されます。

### ステップ 6: オープンループとクローズドループの比較

| 項目 | クローズドループ（Path Follower） | オープンループ（Open Loop） |
|---|---|---|
| ステアリング制御 | 経路追従のため自動調整 | 事前に定義された入力値 |
| 用途 | 経路追従性能の評価 | 車両の過渡応答特性の評価 |
| ドライバーモデル | 必要 | 不要 |
| 再現性 | 経路に依存 | 入力が固定のため高い |

:::caution
オープンループのステアリング入力では、車両が大きく旋回し不安定になる場合があります。シミュレーション時間や走行速度に注意して設定してください。
:::

### ステップ 7: 実行と確認

シミュレーションを実行し、結果のビデオとプロットで車両の挙動を確認します。90° のステアリング入力に対する車両のヨーレート、横加速度、ロール角の応答を観察してください。
