# 演習4: サスペンションパラメータの変更

## 学習目標

- ネストされたパラメータへの **Copy and Link チェーン** を理解する
- サスペンションのばね定数を変更する方法を習得する
- パラメータ変更前後の結果をオーバーレイで比較する
- ばね定数の変化が車両挙動に与える影響を考察する

## 前提条件

- [演習1](./exercise-1.md) および [演習2](./exercise-2.md) を完了していること
- Copy and Link の操作に慣れていること

---

## Copy and Link チェーンについて

CarSim のデータ構造は階層的に構成されています。例えば、ばね定数を変更するには以下のようなチェーンをたどる必要があります:

```
Simulation (Home) → Vehicle Assembly → Front Suspension → Springs, Dampers → Spring Rate
```

途中の各レベルで **Copy and Link** を行うことで、変更したいパラメータだけを独立して編集できます。元のデータセットを直接変更しないため、他のシミュレーションに影響を与えません。

:::info
Copy and Link を使わずに元のデータセットを直接編集すると、そのデータセットを参照しているすべてのシミュレーションに影響が及びます。必ず Copy and Link を使って独立したコピーを作成してから編集してください。
:::

## 手順

### ステップ 1: ベースラインの選択

ツリーから **Quick Start Guide Example → Baseline** に移動します。

### ステップ 2: データセットの複製

Baseline を **Duplicate** し、新しいデータセットを作成します。

- **Category**: `** CS Training`
- **Title**: `#4 Change Front Suspension Spring Rate`

### ステップ 3: Vehicle Assembly の Copy and Link

Vehicle Assembly を右クリックし、**Copy and Link** を選択します。

新しい名前: `C-Class, Hatchback, Front Sus. Spring Rate = 10`

### ステップ 4: フロントサスペンションへの移動

Copy and Link で作成した Vehicle Assembly のリンクをクリックし、**Front Suspension** セクションに移動します。さらに **Springs, Dampers, and Compliance** の設定画面を開きます。

### ステップ 5: ばね・ダンパーデータセットの Copy and Link

ばね・ダンパーのデータセットを右クリックし、**Copy and Link** を選択します。

新しい名前: `Front Sus. Spring Rate = 10`

### ステップ 6: ばね定数の変更

ばね定数（Spring Rate）の値を **10 N/mm** に変更します。

:::caution
通常の乗用車のフロントサスペンションばね定数は 20〜40 N/mm 程度です。10 N/mm は非常に柔らかい設定であり、車両の挙動に大きな変化が生じます。この演習ではパラメータ変更の効果を明確に観察するため、意図的に大きく変更しています。
:::

### ステップ 7: オーバーレイの設定

1. **Home** 画面に戻ります
2. オーバーレイの色を**赤**に設定します
3. **"Overlay videos and plots with other runs"** を有効にします
4. オーバーレイ対象として、元の **Baseline** データセットをリンクします

### ステップ 8: シミュレーションの実行

**Run Math Model → Execute** でシミュレーションを実行します。

### ステップ 9: 結果の比較

**Video + Plot** で結果を確認し、以下の点に注目します:

- **ロール角（Roll Angle）**: 柔らかいばねにより、旋回時のロール角が大きくなります
- **横方向荷重移動**: ロール角の増加に伴い、左右タイヤ間の荷重移動量が変化します
- **ヨーレート応答**: ロール特性の変化がヨーレート応答に与える影響を観察します

---

## 結果の考察

フロントサスペンションのばね定数を標準値から 10 N/mm に下げると、以下の変化が予想されます:

| 項目 | 変化の方向 | 理由 |
|---|---|---|
| ロール角 | 増加 | ばねが柔らかくなり、ロール剛性が低下するため |
| フロント荷重移動 | 減少 | フロントのロール剛性配分が小さくなるため |
| アンダーステア傾向 | 減少 | フロントの横力余裕が増加するため |

:::tip
この演習を応用して、リアサスペンションのばね定数を変更した場合や、フロント・リア両方を変更した場合の挙動も試してみると、ロール剛性配分と操縦安定性の関係をより深く理解できます。
:::
