# ドライビングシミュレーター

CarSim は Driver-in-the-Loop（DiL）シミュレーションに対応しており、人間のドライバーがリアルタイムに車両を操作するドライビングシミュレーターを構築できます。エントリーレベルのゲームコントローラーからエンジニアリンググレードの研究用シミュレーターまで、幅広い環境をサポートしています。

## 対応コントローラー

CarSim は複数のステアリングコントローラーに対応しています。用途と予算に応じて最適なコントローラーを選択できます。

### コントローラー比較

| カテゴリ | 製品例 | 駆動方式 | トルク | 更新レート | 用途 |
|---------|-------|---------|-------|-----------|------|
| **エントリー** | Logicool G29/G920 | ギア/ベルト駆動 | ~2.0–2.5 Nm | 180–400 Hz | 基本的な DiL 検証、デモ |
| **プレミアム** | Fanatec CSL/Podium | ダイレクトドライブ | ~20–25 Nm | 1000 Hz | 高忠実度 DiL、主観評価 |
| **R&D** | MdynamiX | サーボモーター | カスタム | EtherCAT/CAN | エンジニアリング研究開発 |

:::info 公式サポート
Logicool G29（PlayStation 向け）および G920（Xbox/PC 向け）は CarSim が公式にサポートするコントローラーです。追加のドライバーソフトウェアなしで動作します。
:::

## Tabletop DS（卓上ドライビングシミュレーター）

**Tabletop DS** は、Windows PC とステアリングコントローラーのみで構築できるシンプルなドライビングシミュレーターです。

- **追加ソフトウェア不要**: CarSim と Windows の標準機能のみで動作
- **簡易セットアップ**: コントローラーを USB 接続するだけで利用可能
- **主な用途**: クイックデモ、基本的な操舵フィーリングの確認、教育目的

Tabletop DS は本格的な DiL システムの導入前に、CarSim のリアルタイム操作を手軽に体験するための入門的な環境です。

## Simulink との連携

Simulink を組み合わせることで、DiL シミュレーションをさらに拡張できます。

- **カスタム制御ロジック**: ADAS 機能や自動運転アルゴリズムを Simulink で実装し、ドライバーとの協調動作を検証
- **計測・ログ**: Simulink のスコープやログ機能を使い、ドライバー入力と車両応答を詳細に記録
- **ハードウェア連携**: Simulink を介して外部ハードウェア（CAN デバイス、センサーなど）と接続

## Unreal Engine 連携

Unreal Engine を使用して、高品質な3Dビジュアライゼーション環境を構築できます。リアルタイムレンダリングにより、ドライバーに対して没入感のある走行環境を提供します。

- **VS Connect プロトコル**: CarSim、Unreal Engine、Simulink 間のマルチプログラム同期を実現
- **リアルタイム環境**: リアルタイムターゲットと組み合わせた高忠実度 DiL システムの構築が可能

```
ドライバー入力 → CarSim (車両ダイナミクス)
                    ↕ VS Connect
              Unreal Engine (映像出力) → ディスプレイ
                    ↕ VS Connect
              Simulink (制御ロジック)
```

## VI-grade 統合

**VI-grade** は高忠実度ドライビングシミュレーションプラットフォームを提供しており、CarSim との統合が可能です。

### DiM 150 シミュレーター

VI-grade の **DiM 150** は、9自由度（9-DOF）のモーションプラットフォームを備えたフルスケールドライビングシミュレーターです。

| 仕様 | 詳細 |
|------|------|
| 自由度 | 9-DOF |
| モーション | ヘーブ、ロール、ピッチ、ヨー、サージ、スウェイ + 3軸振動 |
| リアルタイムシミュレーション | Concurrent RT |
| CarSim 統合 | cpar ファイル経由 |

### 統合手順

VI-grade との統合に必要な cpar ファイルは、[carsim.com/users/vi-grade](https://carsim.com/users/vi-grade) から入手できます。

:::tip セットアップのポイント
VI-grade 統合では、Concurrent RT をリアルタイムシミュレーションエンジンとして使用します。CarSim モデルは Concurrent RT 上で実行され、VS Connect を介してビジュアライゼーションおよびモーションプラットフォームと同期します。
:::

## VS Connect プロトコル

**VS Connect** は、複数のプログラム間でデータをリアルタイムに同期するための CarSim 独自のプロトコルです。

- CarSim ↔ Unreal Engine 間の車両位置・姿勢データの共有
- CarSim ↔ Simulink 間の制御変数の同期
- 低遅延・高帯域のデータ転送
- 複数クライアントの同時接続に対応

VS Connect により、ドライビングシミュレーター全体を一つの統合システムとして構成し、各コンポーネント間の時刻同期を維持しながらリアルタイムシミュレーションを実行できます。
