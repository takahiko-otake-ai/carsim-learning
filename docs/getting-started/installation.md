# インストールとセットアップ

CarSim を使い始めるために必要な環境構築手順を解説します。

## システム要件

CarSim は **Windows OS** 上で動作するシミュレーションソフトウェアです。以下の要件を満たす PC を用意してください。

- **OS**: Windows 10 / 11（64 ビット）
- **CPU**: Intel Core i5 以上を推奨（マルチコア推奨）
- **メモリ**: 8 GB 以上（大規模シミュレーションでは 16 GB 以上を推奨）
- **ディスク容量**: インストールに約 2 GB、データベースに追加容量が必要
- **GPU**: VS Visualizer での 3D アニメーション表示に DirectX 対応 GPU を推奨
- **ディスプレイ解像度**: 1920×1080 以上を推奨

## インストール手順

1. **インストーラの入手** — [carsim.com](https://www.carsim.com) のダウンロードページからインストーラを取得します。
2. **インストーラの実行** — ダウンロードした `.exe` ファイルを管理者権限で実行し、画面の指示に従ってインストールを進めます。
3. **インストール先の確認** — デフォルトでは `C:\Program Files\CarSim_XX.X\` にインストールされます（`XX.X` はバージョン番号）。

:::tip
複数バージョンを併用する場合、各バージョンは個別のフォルダにインストールされるため共存が可能です。
:::

## ライセンスの設定

CarSim では主に 2 種類のライセンス形態が利用できます。

| 形態 | 説明 |
|------|------|
| **ドングルライセンス** | USB ドングルを PC に接続して認証します。ドングルが接続されている間、ソフトウェアを使用できます。 |
| **ネットワークライセンス** | ライセンスサーバーからネットワーク経由で認証を取得します。複数ユーザーでライセンスを共有する環境に適しています。 |

ライセンスの種類に応じて、初回起動時にドングルの接続確認またはサーバー情報の設定を行ってください。

## データベースの作成

CarSim はシミュレーション設定を「データベース」として管理します。初回起動時に **Database Builder** を使ってデータベースを作成します。

- **Full（フル）**: すべてのサンプルデータセットを含むデータベースを作成します。学習やテスト用途に最適です。
- **Minimal（ミニマル）**: 最小限の構成でデータベースを作成します。特定の用途に絞った軽量なセットアップが可能です。

:::info
初めて CarSim を使う場合は **Full** を選択することをお勧めします。豊富なサンプルを使って各機能を学ぶことができます。
:::

## 初回起動

インストールとデータベース作成が完了したら、CarSim を起動します。起動すると **Select Recent Database** ウィンドウが表示され、作成済みのデータベース一覧から使用するデータベースを選択できます。

データベースを選択すると、CarSim の Run Control 画面（ホーム画面）が開き、シミュレーションを始める準備が整います。
