# 最初のシミュレーション

CarSim をインストールしたら、実際にシミュレーションを実行してみましょう。ここでは Quick Start サンプルを使って、基本的なワークフローを体験します。

## Quick Start サンプルを開く

CarSim を起動すると **Run Control** 画面が表示されます。これがシミュレーションのホーム画面であり、すべての操作の起点となります。

Run Control 画面には、現在選択されているデータセットの概要が表示されています。初めて使う場合は、データベースに含まれる **Quick Start** カテゴリのサンプルを開いてください。Library Browser からサンプルを選択するか、Run Control のドロップダウンから Quick Start 例題を選びます。

## Run Control 画面の構成

Run Control 画面では、シミュレーションに必要な主要設定へのリンクが集約されています。

- **車両（Vehicle Assembly）** — 車両モデルの設定
- **走行手順（Procedure）** — 操舵・加減速などのドライバー入力
- **道路（Road）** — 走行する道路環境
- **その他の設定** — 可視化、出力変数など

各項目をクリックすると、対応するライブラリの詳細設定画面に遷移できます。

## シミュレーションの実行

設定を確認したら、Run Control 画面の **Run Math Model** ボタンをクリックします。CarSim のソルバーが起動し、シミュレーション計算が実行されます。計算時間はシナリオの長さや複雑さによって異なりますが、一般的な走行シナリオであれば数秒で完了します。

:::tip
Run Math Model ボタンの横に表示されるプログレスバーで、計算の進捗を確認できます。
:::

## 結果の確認

計算が完了したら、**Video + Plot** ボタンをクリックして結果を確認します。VS Visualizer が起動し、アニメーションウィンドウとプロットウィンドウが表示されます。

### VS Visualizer の基本

VS Visualizer は CarSim の結果可視化ツールで、次の 2 つのウィンドウで構成されます。

- **アニメーションウィンドウ** — 車両の走行を 3D で再現します
- **プロットウィンドウ** — 速度、操舵角、横加速度などの時系列データをグラフ表示します

### アニメーション操作

アニメーションウィンドウでは、マウス操作でカメラを自由にコントロールできます。

| 操作 | 動作 |
|------|------|
| 左ドラッグ | カメラの回転 |
| 右ドラッグ | カメラの平行移動 |
| スクロールホイール | ズームイン・ズームアウト |

### プロット操作のキーボードショートカット

プロットウィンドウでは、以下のキーボードショートカットが利用できます。

| キー | 機能 |
|------|------|
| `Z` | プロットを最大化表示 |
| `R` | 表示範囲をリセット |
| `V` | 数値表示の切り替え |
| `G` | グリッド線の表示/非表示 |
| `Space` | 再生/一時停止 |

:::info
これらのショートカットを覚えると、結果の分析を効率的に行えます。特に `Z` キーで個別のプロットを拡大して詳細を確認する操作は頻繁に使います。
:::

## 次のステップ

最初のシミュレーションが成功したら、車両パラメータや走行シナリオを変更して、結果の違いを観察してみてください。パラメータの変更方法は後続のセクションで詳しく解説します。
